C

How active layers
are chosen

1

topk

Restrict to the highest ⌈1.75Na⌉ layers by imp(l), then sample Na.

2

noisy topk

Add Gaussian noise with σ = 0.30, then take the top Na.

3

weighted

Sample without replacement using weights imp(l) · exp(0.20·𝒩(0,1)).

4

random

Ignore the prior and return a uniform random subset of size Na.

Modes differ in how strongly they trust the importance prior.

D

Candidate construction for
a grid cell (Na, φH, mode)

Select active set A ⊂ {1,…,L} using the chosen subset mode.
Assign noisy weights wl = imp(l)·exp(0.35ξl) and normalize to w̃l.
Split budget into attention and feed-forward shares: φHB and (1−φH)B.
For each l ∈ A, compute al from the head share and clip to [1,H].
For each l ∈ A, compute nl from the neuron share and clip to [bn,N].
Sample retained heads using softmax(sH[l,·]/τ), with τ ∼ Unif[0.55,1.10].
Set inactive layers outside A to al = nl = 0.
